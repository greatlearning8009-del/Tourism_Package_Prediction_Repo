# Data folder

Drop `tourism.csv` into this folder before running the pipeline or the
app.

Expected filename: `tourism.csv`

Expected columns: `CustomerID`, `ProdTaken` (target), `Age`,
`TypeofContact`, `CityTier`, `DurationOfPitch`, `Occupation`, `Gender`,
`NumberOfPersonVisiting`, `NumberOfFollowups`, `ProductPitched`,
`PreferredPropertyStar`, `MaritalStatus`, `NumberOfTrips`, `Passport`,
`PitchSatisfactionScore`, `OwnCar`, `NumberOfChildrenVisiting`,
`Designation`, `MonthlyIncome`.

The `splits/` subfolder is generated by `prep.py` and is gitignored.
